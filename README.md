This is a vibe code experiment with Qwen 3 Coder 30b to prototype a web interface. It is in no shape or form production ready. Use at your own risk.

This game map viewer includes all the requested functionality:

1. **Map Display**: Shows a procedurally generated map with terrain, rivers, roads, and locations
2. **Panning**: Click and drag to move around the map
3. **Zooming**: 
   - Mouse wheel zooming
   - Zoom in/out buttons
   - Reset view button
4. **Location Pins**: 
   - Colorful pins at various locations
   - Information popups when clicking on pins
   - Location types and descriptions

## How to Use

1. **Pan the map**: Click and drag to move around
2. **Zoom**: Use mouse wheel or zoom buttons
3. **View locations**: Click on any pin to see details
4. **Reset view**: Use the reset button to return to the starting position

The interface is responsive and will adjust to different screen sizes. The map features a fantasy-style landscape with various terrain elements and locations to explore.
